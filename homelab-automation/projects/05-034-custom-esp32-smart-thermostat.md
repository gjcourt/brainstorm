# Custom ESP32 Smart Thermostat

## Description
Build a custom thermostat using an ESP32, a temperature/humidity sensor (like the BME280), and relays to control your HVAC system. Implement a PID loop for precise temperature control and integrate it with Home Assistant via MQTT.
