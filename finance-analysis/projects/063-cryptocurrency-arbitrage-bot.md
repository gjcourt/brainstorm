# Cryptocurrency Arbitrage Bot

## Description
Write a bot in Go that connects to multiple cryptocurrency exchanges (e.g., Binance, Coinbase, Kraken) simultaneously. Monitor order books for price discrepancies across exchanges and execute triangular or spatial arbitrage trades when profitable.
