# Insider Trading Tracker and Alert System

## Description
Build a scraper that monitors the SEC EDGAR database for Form 4 filings (Statement of Changes in Beneficial Ownership). Parse the data to identify significant insider buying or selling and send real-time alerts to a Slack or Discord channel.
