# Peer-to-Peer Lending Portfolio Analyzer

## Description
If you use P2P lending platforms (like Prosper or LendingClub), build a tool that downloads your loan portfolio via their API. Analyze the default rates, calculate your true annualized return (XIRR), and visualize the risk distribution of your notes.
