# Espresso Recipe and Extraction Database

## Description
Create a personal database to log your daily espresso shots. Track the bean origin, roast date, dose, yield, extraction time, temperature, and a subjective taste score (e.g., acidity, sweetness, body) to dial in new coffees faster.
