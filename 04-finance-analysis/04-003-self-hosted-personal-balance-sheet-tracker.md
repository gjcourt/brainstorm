# Self-Hosted Personal Balance Sheet Tracker

## Description

Build a custom, self-hosted application to track your personal net worth. Integrate with the Plaid
API to pull live bank and brokerage balances, categorize assets and liabilities, and calculate
personal financial ratios (e.g., debt-to-equity).
