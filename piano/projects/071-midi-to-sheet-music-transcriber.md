# MIDI to Sheet Music Transcriber

## Description
Build a tool that takes a raw, unquantized MIDI performance (e.g., an improvisation session) and intelligently transcribes it into readable sheet music. Implement algorithms to guess the time signature, key signature, and separate the left/right hand parts.
