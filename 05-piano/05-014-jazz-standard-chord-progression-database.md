# Jazz Standard Chord Progression Database

## Description
Create a structured, searchable database (e.g., in JSON format) of the chord progressions for the top 100 jazz standards. Build a simple frontend to quickly pull up the changes for a song, transpose them to any key, and display Roman numeral analysis.
