# Go-based DSP Audio Effects Engine

## Description

Write a real-time audio effects engine from scratch in Go. Implement basic effects like delay,
reverb, and chorus using ring buffers and mathematical algorithms, and expose them via a VST/AU
plugin wrapper or a standalone JACK client.
